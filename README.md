# Ratchet: Deadlocked Manual AP World Randomizer

This randomizer is intended for use with the Archipelago randomizer system using the Manual Client installation. It currently uses any version of Ratchet: Deadlocked.
## How To Use

1.) Install the latest stable Manual Client from the [Manual For Archipelago](https://github.com/ManualForArchipelago/Manual/releases) GitHub release page.
This is installed same as any other apworld in your `custom_worlds` folder.

2.) Install the `manual_ratchetdeadlocked_amondo.apworld` in your `custom_worlds` folder and your modified yaml into your `Players` folder. In order for the Manual Client to work properly, the yaml will need to be kept in this folder during your session.

3.) Open your Archipelago Launcher and click `Generate.` Upload, or host, your generated seed.

4.) Open the Manual Client in your Archipelago Launcher and enter your server details. If successful, it will connect and the `Manual` tab will populate with game-specific data.
- Additionally, if you have the Universal Tracker AP World installed, all locations that are considered in logic will be highlighted.
